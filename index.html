Что это дает?

Полноценное Unix-подобное рабочее окружение на вашей Windows

Тайлинговый оконный менеджер

Как в Linux (i3, hyprland). Окна автоматически располагаются в плитках без перекрытия.

Верхняя панель

Системная информация, время, сеть, батарея и пользовательские виджеты как в GNOME.

Док-панель

Заменитель панели задач Windows. Чистый рабочий стол без иконок как в macOS.

Горячие клавиши

Полное управление окнами с клавиатуры. Vim-стиль навигации (HJKL).

Единое оформление

Согласованная темная тема Catppuccin Mocha для всех компонентов системы.

Быстрый запуск

Flow Launcher для мгновенного поиска и запуска приложений по Alt+Q.

Компоненты системы

TilingForWindows объединяет лучшие инструменты для создания идеального рабочего окружения

GlazeWM

Тайлинговый оконный менеджер для Windows. Организует окна в автоматически управляемые плитки. Аналогичен i3wm в Linux.

Основные возможности:

  • Автоматический и ручной тайлинг
  • 9 рабочих пространств
  • Vim-навигация (HJKL)
  • Конфигурация через YAML
GitHub

Yasb

Настраиваемая верхняя панель статуса. Отображает системную информацию, время, сеть, батарею и пользовательские виджеты.

Особенности:

  • Модульная архитектура
  • Поддержка плагинов
  • Тема Catppuccin Mocha
  • Требуются Nerd Fonts
GitHub

Nexus

Продвинутая док-панель для Windows. Заменяет стандартную панель задач и скрывает иконки рабочего стола.

Возможности:

  • Полностью настраиваемая
  • Анимации и эффекты
  • Группы приложений
  • Системный трей
Сайт

Flow Launcher

Мощный и быстрый лаунчер для Windows. Мгновенный поиск приложений, файлов и выполнение команд.

Особенности:

  • Запуск по первым буквам
  • Поиск файлов и папок
  • Встроенный калькулятор
  • Поддержка плагинов
Сайт

Установка TilingForWindows

Быстрая автоматическая установка или ручная настройка компонентов

1

Автоматическая установка (рекомендуется)

Скачайте установщик TilingForWindows и запустите его от имени администратора.

Скачать TilingForWindows.exe
2

Ручная установка компонентов

Если автоматическая установка не работает, установите компоненты вручную:

# 1. GlazeWM (тайлинговый менеджер)
https://github.com/glzr-io/glazewm/releases/download/v3.9.1/glazewm-v3.9.1.exe

# 2. Yasb (верхняя панель)
https://github.com/amnweb/yasb/releases/download/v1.8.6/yasb-1.8.6-x64.msi

# 3. Nexus (док-панель)
https://www.winstep.net/nexus.zip

# 4. Flow Launcher (быстрый запуск)
https://github.com/Flow-Launcher/Flow.Launcher/releases/download/v2.0.3/Flow-Launcher-Setup.exe

# 5. Шрифты Nerd Fonts (для иконок в Yasb)
https://github.com/ryanoasis/nerd-fonts/releases/latest/download/JetBrainsMono.zip
3

Настройка после установки

Если верхняя панель (Yasb) не показывает иконки, установите JetBrainsMono Nerd Font.

# В папке Fonts должны быть файлы:
JetBrainsMonoNerdFont-Regular.ttf
JetBrainsMonoNerdFont-Bold.ttf
# и другие начертания

Если конфиги не установились автоматически, скопируйте их вручную:

cd C:\Apps\TilingForWindows
xcopy /Y /E glazewm\* "%USERPROFILE%\.glzr\glazewm\"
xcopy /Y /E yasb\* "%USERPROFILE%\.config\yasb\"
nexus_config.reg
